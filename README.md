# 📝 Simple Registration Form

A clean, responsive registration form built with **HTML**, **Tailwind CSS**, and a lil bit of **JavaScript** ⚡

## 🚀 Features
- 👤 Username field
- 🔒 Password field
- 📧 Email field
- ✅ Register button with basic validation

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS
- JavaScript

## 📂 How to Use
1. Clone this repo 📥
2. Open `index.html` in your browser 🌐
3. Fill the form and hit submit ✅
